# SaymoShop

1. Clone project
2. ``` npm install ```
3. Clone file ``` .env.template ``` and rename to ``` .env ```
4. Change the enviroment variables
5. Set up the data base
```
docker-compose up -d
```

6. Execute Seed

```
GET localhost:3000/seed/seed
```

7. Wake up: ``` npm run start:dev ```
